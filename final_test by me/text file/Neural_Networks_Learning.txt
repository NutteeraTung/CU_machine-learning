# สรุปไฟล์ Neural_Networks_Learning.ipynb

## 1. บทนำ

ไฟล์นี้เป็นแบบฝึกหัดเกี่ยวกับการเรียนรู้ของโครงข่ายประสาทเทียม (Neural Networks) โดยเน้นที่การใช้อัลกอริทึม Backpropagation เพื่อคำนวณ Gradient สำหรับCost function  (Cost Function) ของโครงข่ายประสาทเทียม และนำไปประยุกต์ใช้กับงานจดจำตัวเลขที่เขียนด้วยลายมือ

## 2. การแสดงข้อมูล

ข้อมูลที่ใช้เป็นภาพตัวเลขขนาด 20x20 พิกเซล ซึ่งถูกแปลงเป็นเวกเตอร์ขนาด 400 มิติ โดยมีตัวอย่างทั้งหมด 5,000 ตัวอย่าง

## 3. โครงสร้างโครงข่ายประสาทเทียม

โครงข่ายประสาทเทียมที่ใช้มี 3 ชั้น:
- ชั้นอินพุต: 400 โหนด (จากภาพขนาด 20x20 พิกเซล)
- ชั้นซ่อน: 25 โหนด
- ชั้นเอาต์พุต: 10 โหนด (สำหรับตัวเลข 0-9)

## 4. Cost function และ Feedforward

ในส่วนนี้มีการสร้างฟังก์ชัน Sigmoid และCost function สำหรับโครงข่ายประสาทเทียม โดย Cost function คำนวณจาก:
- การทำ Feedforward ผ่านโครงข่าย
- การคำนวณค่าความผิดพลาดระหว่างผลลัพธ์กับค่าที่ต้องการ
- การเพิ่มเทอม Regularization เพื่อป้องกันการ Overfitting

## 5. Backpropagation

ส่วนนี้เป็นการพัฒนาอัลกอริทึม Backpropagation เพื่อคำนวณ Gradient ของพารามิเตอร์ในโครงข่าย โดยมีขั้นตอนดังนี้:
1. คำนวณค่าความผิดพลาดที่ชั้นเอาต์พุต
2. ส่งค่าความผิดพลาดย้อนกลับไปยังชั้นซ่อน
3. คำนวณ Gradient ของพารามิเตอร์แต่ละตัว
4. เพิ่มเทอม Regularization ให้กับ Gradient

## 6. การตรวจสอบ Gradient

มีการตรวจสอบความถูกต้องของ Gradient ที่คำนวณได้จาก Backpropagation โดยเปรียบเทียบกับการคำนวณ Gradient ด้วยวิธีเชิงตัวเลข (Numerical Gradient)

## 7. การเรียนรู้พารามิเตอร์

ใช้ฟังก์ชัน `optimize.minimize` จาก SciPy เพื่อหาค่าพารามิเตอร์ที่เหมาะสมที่สุดสำหรับโครงข่ายประสาทเทียม โดยสามารถปรับค่า:
- `maxfun`: จำนวนรอบสูงสุดในการหาค่าที่เหมาะสม
- `lambda_`: ค่าพารามิเตอร์ Regularization

## 8. ผลลัพธ์

หลังจากการเรียนรู้ โครงข่ายประสาทเทียมสามารถจำแนกตัวเลขได้ด้วยความแม่นยำสูง (มากกว่า 95%) บนชุดข้อมูลฝึกฝน

## 9. การแสดงผลชั้นซ่อน

มีการแสดงภาพของน้ำหนักในชั้นซ่อน ซึ่งแสดงให้เห็นว่าโครงข่ายได้เรียนรู้รูปแบบต่างๆ ของเส้นและลักษณะเฉพาะของตัวเลข

## 10. แบบฝึกหัดเพิ่มเติม (ทางเลือก)

ส่วนนี้เป็นการทดลองปรับค่าพารามิเตอร์ต่างๆ เช่น ค่า Regularization และจำนวนรอบในการเรียนรู้ เพื่อดูผลกระทบต่อประสิทธิภาพของโครงข่ายประสาทเทียม

โดยสรุป ไฟล์นี้แสดงการพัฒนาโครงข่ายประสาทเทียมตั้งแต่การสร้างCost function  การคำนวณ Gradient ด้วย Backpropagation การหาค่าพารามิเตอร์ที่เหมาะสม และการประเมินผลประสิทธิภาพของโมเดล
